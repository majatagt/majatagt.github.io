Unit 11 learning outcomes

Upon completion of the unit the student will be able to:

System implementation assignment

This is a development exercise where you are required to implement an object-oriented system. You are acting as a consultant software developer for a local doctors' surgery, who have had a system design created by a software architect. It is your job to implement their design using an object-oriented approach, using Python.

Instructions


The deliverables for this assignment and the associated grading criteria (to be reviewed alongside the criteria grid in the Module Resources page) are: Python code, covering:
・Code elegance - is the appropriate approach (techniques, algorithms etc) used? and
・Meeting the given requirements (Knowledge and Understanding weighted at 30%)
・Application of object-oriented programming features and
・Evidence of testing (Application of Knowledge and understanding weighted a 30%)
・Structure and Presentation (weighted at 30%) focuses on:
・comments on the code explaining it,
your README file containing a description of the design implementation and instructions on how to execute the code, as well as the commentary requested above, and how well you have organised your code Please note that academic integrity also applies to codes, where all sources can be placed in the README file and/or as part of the code commentary. (Academic Integrity weighted at 10%)
